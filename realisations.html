Application GSB

AP 2024-2025

📋 Contexte

Projet de 2ème année de BTS SIO réalisé dans le cadre des Ateliers de Professionnalisation. L'objectif était de développer une application de gestion des frais pour l'entreprise GSB (Galaxy Swiss Bourdin).

🎯 Objectifs

  • Permettre aux visiteurs médicaux de saisir leurs frais
  • Automatiser la validation et le remboursement des frais
  • Gérer les droits d'accès (visiteurs, comptables, administrateurs)
  • Générer des rapports et statistiques

⚙️ Technologies utilisées

Backend: PHP 8, Symfony 6
Frontend: Twig, Bootstrap 5
Base de données: MySQL 8
Outils: Git, Composer, XAMPP

💡 Réalisations techniques

  • Architecture MVC avec Symfony
  • Système d'authentification sécurisé
  • CRUD complet pour la gestion des frais
  • Upload et gestion de fichiers justificatifs
  • Génération de PDF pour les rapports
  • Tests unitaires avec PHPUnit

🎓 Compétences mobilisées

B1.2 - Répondre aux incidents et demandes
B1.4 - Travailler en mode projet (méthode Agile)
B1.5 - Mise à disposition (tests, déploiement, documentation)

📊 Résultats

Application fonctionnelle déployée et testée. Gain de temps de 60% pour le traitement des frais. Documentation complète fournie.
